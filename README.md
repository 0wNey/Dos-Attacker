# Dos-Attacker
Un outil simple et rapide d'utilisation pour effectuer une attaque DoS

Usage: 

________   ________    _________ 
\______ \  \_____  \  /   _____/ 
 |    |  \  /   |   \ \_____  \  
 |    `   \/    |    \/        \ 
/_______  /\_______  /_______  / 
        \/         \/        \/  attacker

[Options]:
		
				-t : target_IP
				-p : port
Exemple : python Dos_attacker.py -t 192.168.1.1 -p 80
